# Get Latest Activity

{% hint style="warning" %}
This endpoint requires `Authorization: Bearer <api key>`.
{% endhint %}

{% hint style="info" %}
Recommended safe rate limit: `27 requests/min` per credential.

The Kong gateway is configured slightly higher than these values to leave a small safety buffer for normal gameplay bursts.
{% endhint %}

Returns the latest clock-in or activity record.

#### Request

- Query parameter `type` selects `clockin` or `activity`.
- Optional `serverId` and `clockInType` filters are supported.

#### Response

- Returns the latest activity item for the selected mode.
